# project_kickoff.md 템플릿

이 파일은 `kickoff-profile` 스킬이 project_kickoff.md를 생성할 때 참조하는 표준 구조다.
각 섹션의 헤더와 항목은 이후 스킬(kickoff-gap, kickoff-checklist)이 파싱하므로 **헤더 포맷을 변경하지 말 것**.

---

## 작성 규칙

1. **프로젝트명**: 사용자가 명시하지 않으면 AI가 아이디어에서 적절한 이름 생성
2. **AI 추천 표기**: 인터뷰에서 "추천해줘"로 답한 항목은 값 뒤에 `(AI 추천)` 표기
3. **빈 항목 처리**: 정보가 없는 항목은 "미정" 표기 (빈칸으로 두지 않음)
4. **섹션 5, 6 플레이스홀더**: 반드시 헤더만 남겨두기 (이후 스킬이 내용 추가)
5. **설계 패스 시**: 섹션 2, 3, 4를 통째로 생략 (헤더도 없음)
6. **언어**: 사용자가 사용한 언어로 작성 (한국어 기본)
