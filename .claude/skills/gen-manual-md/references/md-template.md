## 작성 규칙

- 목차 앵커: 소문자, 공백은 `-`, 특수문자 제거 (예: `#1-개요`)
- 영어 앵커: `#1-overview`
- 표: GitHub Flavored Markdown 호환 (`|---|` 구분선 필수)
- 경고: `> ⚠️ **주의/Warning:**` blockquote 형식
- FAQ: 볼드 Q + 일반 A (details 태그 미사용 — 호환성 우선)
- 조건부 섹션 제거 시 번호 재조정 필수
- 푸터 한 줄로 생성일 표시
